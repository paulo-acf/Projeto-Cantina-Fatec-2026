O Sistema de Cantina é um projeto desenvolvido em Python com o objetivo de gerenciar de forma simples:
   •O estoque;
   •As vendas;
   •O consumo de produtos.
O sistema permite cadastrar produtos com:
   •Nome;
   •Preço de compra e venda;
   •Datas;
   •Quantidade disponível; 
   •Controle automáto de estoque. 
Também é possível, tanto no terminal quanto em formato PDF, gerar relatórios de:
   •Vendas;
   •Consumo; 
   •Estoque; 
   •Avisos.
O sistema pode ser utilizado com dados inseridos manualmente ou com geração automática utilizando a biblioteca Faker, sendo recomendado escolher apenas um dos modos para manter a organização.
